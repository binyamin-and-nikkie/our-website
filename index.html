Binyamin's Notes

    Frontend vs Backend

  • front end is everything the users sees & interacts with (client)
  • backend is everything else "behind the scenes" - like keeping of data, users, etc. (server)
  • Static websites stays the same - like Discount Luggage’s website - it’s only Frontend, just for informational purposes
  • Dynamic websites contain a database,users, interaction, etc. (like JewishBookCorner). It - has a backend)

    HTML

  • HyperText Markup Language
  • Structure of a webpage
  • “Nouns” of the webpage
  • HTML elements have opening and closing tags

    CSS

  • It defines the style and design of html
  • “Adjectives” of the webpage

    Javascript

  • It adds logic and interaction
  • “Verbs” of the webpage (defines all the actions)
